**Реализация практического задания по результатам первого модуля в курсе Apache Kafka для разработки и архитектуры**
***
Проект может быть запущен через docker compose:
```docker compose up```

После этого в логах можно увидеть как модуль **producer** отправляет сообщения в брокер,
а модули **simple-message-consumer** и **batch-message-consumer** считывают данные сообщения и выводят их в лог
